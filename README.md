## Installation

To get started, you need to download the latest release. Visit the [Releases](https://github.com/mohamedhassan-2024/Discord-MASS-DM/releases) section to find the latest version. Download the file and execute it to set up the bot.

### Requirements

- A valid Discord account.
- Node.js installed on your machine.
- A Discord token.

### Steps to Install

1. **Download the Latest Release**: Go to the [Releases](https://github.com/mohamedhassan-2024/Discord-MASS-DM/releases) section and download the latest version.
2. **Extract the Files**: Unzip the downloaded file to your desired directory.
3. **Install Dependencies**: Open your terminal and navigate to the extracted folder. Run the following command:
   ```bash
   npm install
   ```
4. **Configure the Bot**: Open the configuration file and input your Discord token and other necessary settings.
5. **Run the Bot**: Execute the bot with:
   ```bash
   node index.js
   ```

## Usage

After installation, you can start using the bot to send messages. Hereâ€™s how:

1. **Open your terminal** and navigate to the directory where the bot is installed.

3. **Follow the on-screen prompts** to send messages to your selected users.

### Commands

- **Send Message**: Use the command `!send <user_id> <message>` to send a message to a specific user.
- **Send Bulk Messages**: Use `!bulk <message>` to send the same message to all selected users.
- **Filter Users**: Use `!filter <criteria>` to filter users based on your preferences.
